# Team Excelsior

A Next.js + Tailwind team operations app with dashboard, team directory, sessions, progress tracking, and settings.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Routes

- `/` Dashboard
- `/team` Team directory
- `/sessions` Sessions
- `/progress` Progress tracker
- `/settings` Settings

## SQL setup

Run SQL in `sql/progress_tables.sql` in Supabase SQL editor.
